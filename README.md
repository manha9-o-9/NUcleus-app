# NUcleus — Setup Guide

## First time setup

Open a terminal in the `nucleus-app` folder and run:

```
npm install
cd backend && npm install && cd ..
npm start
```

## After first time (just run this every time)

```
npm start
```

## What you'll see in the terminal

```
[server] 🚀 NUcleus backend running → http://localhost:5000
[client] VITE ready → http://localhost:5173
```

Open → http://localhost:5173
